## Os operadores relacionais: 
- == igualdade.
- != diferente.
- > maior que.
- < menor que.
- >= maior ou igual que.
- <= menor ou igual que.
## Operadores lógicos:
- AND: só será "true" quando as duas expressões forem verdade
- OR: só será "false" quando as duas expressões forem falsas
- NOT: será falso quando a expressão for verdadeira e vice-versa
